# ping-pong-ball led matrix backpack

This is a 5x10 ping-pong ball display, like [this one](https://hackaday.com/2012/07/27/a-much-larger-rainbow-board-of-many-ping-pongs/), that you can ware on you back, or mount on your wall.

It is powered by a xioa esp32c3, ws2812b LEDs and a usb power bank.
